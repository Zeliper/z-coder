# LESSONS LEARNED

빌드 에러 및 해결 방법을 기록하는 파일입니다. builder-agent가 자동으로 업데이트합니다.

## 사용 방법

- 새로운 에러 패턴 발견 시 아래 형식으로 추가
- coder-agent는 작업 전 이 파일을 참조하여 알려진 문제 회피
- 프로젝트별로 누적되는 학습 데이터

### 예시 형식

```markdown
## [에러 유형] 에러 제목

- 조건 1
- 조건 2

### 에러 메시지
```
에러 메시지 내용
```

### 원인
원인 설명

### 해결 방법
1. 해결 단계 1
2. 해결 단계 2

### 예방 방법
- 예방 조치 1
- 예방 조치 2

### 관련 파일
- 파일 경로 1
- 파일 경로 2

### 발생일
YYYY-MM-DD

```
